# Publishing

## Production

Building the production version requires a signing key.

In Android Studio, go to `Build > Generate Signed Bundle / APK` and select `APK` as the package type. Then, select the `release` build type and click `Next`. Finally, select the signing key and click `Finish`. The APK will be generated in the `app/build/outputs/apk/release` directory.

## Testing

Does not require a signing key.
